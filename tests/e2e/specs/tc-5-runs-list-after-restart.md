# TC-5. Список ранов после перезапуска — US-3

1. Создать 2–3 рана подряд с разными заголовками.
2. Закрыть Extension Development Host.
3. Запустить заново (`F5`), открыть ту же папку.

**Ожидание:** список содержит те же раны, свежий — сверху. Даты и статусы совпадают с тем, что было до перезапуска.
