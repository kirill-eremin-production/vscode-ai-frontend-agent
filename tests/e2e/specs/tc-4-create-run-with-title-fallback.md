Имитация: задать заведомо неверный ключ (любая строка) либо отключить интернет.

1. Ввести запрос `Test fallback`, Start run.

**Ожидание:**

- Ран всё равно создан.
- `meta.title` равен исходному запросу (или его первым 40 символам + `…`), а не пустая строка.
- В UI ошибка не показывается (заголовок просто менее красивый).
