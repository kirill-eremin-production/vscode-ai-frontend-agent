# TC-1. Установка ключа OpenRouter через команду — US-1

1. Палитра → `AI Frontend Agent: Set OpenRouter API Key`.
2. Ввести валидный ключ `sk-or-...`, Enter.

**Ожидание:**

- Уведомление «OpenRouter API key сохранён».
- В Activity Bar открыть AI Frontend Agent — кнопка «Set API key» при создании рана больше не вызывает input box (если ключ ещё не запрошен — кейс TC-2 проверит это).
