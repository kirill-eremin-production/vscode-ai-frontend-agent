# TC-2. Установка ключа через кнопку «Set API key» — US-1

1. Удалить ключ (палитра → `AI Frontend Agent: Set OpenRouter API Key` → ввести пустую строку).
2. Открыть сайдбар расширения.
3. Нажать кнопку «Set API key» под textarea.

**Ожидание:** появляется тот же input box. После сохранения дальнейшие запуски ранов идут без повторного вопроса.
