# Smoke checklist

Минимум, что нужно прокликать перед коммитом, если правка касалась рантайма расширения. Полный список тест-кейсов — в [e2e/specs/](e2e/specs/) (`.spec.ts` — автоматизированные, `.md` — ручные).

- [ ] `npm run lint && npm run build` зелёные.
- [ ] `F5` поднимает Extension Development Host без ошибок в `Debug Console`.
- [ ] Иконка «AI Frontend Agent» видна в Activity Bar.
- [ ] Создание нового рана с валидным ключом → появляется `.agents/runs/<id>/meta.json` с осмысленным заголовком и `chat.jsonl` с одним сообщением.
- [ ] Список ранов слева отрисовался; клик переключает карточку справа.
- [ ] Кнопка «Open in tab» открывает панель и не дублирует вкладки при повторном нажатии.
- [ ] Запуск без открытой папки даёт понятную ошибку, а не падение.
- [ ] Отмена ввода ключа не создаёт ран и не показывает ошибку.

Если хоть один пункт красный — фиксим до коммита.
